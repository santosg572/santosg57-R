R como calculadora
==================

**Constantes Numericas**

* **pi** - equivale 3.1415.....

* **Inf** - es el infinito y prepresenta un número muy grande.


**Operadores Aritméticos**: +, -, *, ^, /, 

Si x, y son números naturales entonces:

* :math:`x \%\% y` calcula el residio de x entre y

* :math:`x \%/\% y` calcula el cociente de x entre y

**Constantes Lógicas**: FALSE, TRUE

**Opreradores de Comparación**: :math:`<, \leq, >, \geq, ==, !=`

**Ejercicios 01.**

1. Calcule las siguiente expresión

.. math::

   \frac{1}{1+ \frac{1}{1+ \frac{1}{1+ \frac{1}{1+ 2}}} }

2. Calcule el peso molecular del agua (:math:`H_2O`).

3. Sobre los extremos de un segmento AB de 1m de longitud se fijan dos cargas.
Una :math:`q_1 = +4 \times 10^{-6}` C, sobre el punto A y otra :math:`q_2 = 1 \times 10^{-6}` C,
sobre el punto B. En que posición del segmento debo de poner una carga :math:`+2 \times 10^{-6}` C
para que este en equilibrio bajo la acción de las dos cargas dadas. 

3. Encuentre la solución de las siguientes ecuaciones:

a) :math:`2x^2 - 3x +1 = 0`

b) :math:`x^2 + x - 1 = 0`
 
c) :math:`x^2 + x + 1 = 0`
